Game Installation Steps
You can download any version of the game for free from both the official website and our GitHub repository.
Depending on your platform, the files will be available as .exe, .apk, or inside a .zip archive that must be extracted before running the game.

Available Formats
Windows: .exe executable

Android: .apk installable package

Multiplatform: .zip archive containing all required files

Installation
Download the version that matches your device.

If the file is compressed, extract it to a folder of your choice.

Run the main game file (.exe on Windows or .apk on Android).

Enjoy playing Geometry Kirk!

Having trouble installing the game?
If any installation step doesn’t work correctly, an error appears, or you’re unable to launch the game, feel free to contact us and we’ll be happy to help.
